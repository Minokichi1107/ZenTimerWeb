## Overview
A timer that uses the sound of a wooden fish (mokugyo), mainly designed as a cooking timer.

## Features
- Time setting
- Volume control
- Timer function
- Settings configuration
  - Number of bell rings
  - Ring duration (seconds)
  - Background operation
